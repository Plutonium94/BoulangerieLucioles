Ceci est le code d'un site web d'une boulangerie imaginaire.
Il est construit avec Node.js, Express en backend avec une base de données en MongoDB.
Côté front il utilise AngularJS.

La base de données n'est pas inclus dans ce repository. Vous pouvez installer MongoDB.

npm install monk
mongod -dbpath {chemin vers dossiers db}
node app.js
